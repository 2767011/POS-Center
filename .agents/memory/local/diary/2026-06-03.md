## Ошибка push в GitHub

- Найдено: GitHub отклоняет `git push origin main:main`, потому что `TSPIoT/esmpack_windows_1.6.2.1.74.zip` имеет размер 137.08 MB и превышает лимит GitHub 100 MB.
- Важно: файл уже находится в локальном коммите `48aa5fb`, поэтому `.gitignore` или простое удаление файла без переписывания локальной истории не исправит push.
- Сделано: добавлен `.gitignore` с правилом `TSPIoT/`; правило проверено через `git check-ignore -v --no-index`.
- Текущее состояние: папка `TSPIoT` перенесена пользователем из рабочей директории, Git видит удаления файлов, но push всё ещё требует удаления `TSPIoT/` из локальной истории коммитов.
- Сделано: два локальных коммита поверх `origin/main` пересобраны в один новый коммит `ff3c9b2` с сообщением `DFU`; `TSPIoT/` не входит в diff, файлов больше 100 MB в `HEAD` не найдено.
